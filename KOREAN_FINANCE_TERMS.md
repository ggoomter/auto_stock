## 📊 주요 지표 용어

### 위험 관리 지표

| 영문 용어 | 한글 용어 | 설명 | 사용 예시 |
|----------|----------|------|----------|
| **Sharpe Ratio** | **위험조정수익률** | 위험 대비 수익의 효율성을 나타내는 지표 | "위험조정수익률이 1.5로 우수합니다" |
| **VaR (Value at Risk)** | **위험가치** | 특정 신뢰수준에서의 최대 예상 손실액 | "95% 신뢰수준 위험가치는 -3%입니다" |
| **CVaR (Conditional VaR)** | **조건부 위험가치** | VaR를 초과하는 극단적 손실의 평균값 | "조건부 위험가치는 -5%입니다" |
| **Maximum Drawdown** | **최대낙폭** | 고점 대비 최대 하락률 | "최대낙폭이 -15%를 초과했습니다" |
| **Beta** | **시장민감도** | 시장 변동에 대한 민감도 | "시장민감도가 1.2로 시장보다 변동성이 큽니다" |
| **Correlation** | **상관관계** | 자산 간의 움직임 연관성 | "두 종목의 상관관계는 0.7입니다" |
| **HHI (Herfindahl Index)** | **집중도지수** | 포트폴리오 종목 편중도 | "집중도지수가 0.3을 초과하면 위험합니다" |
| **Liquidity Score** | **유동성 점수** | 거래 활성도 및 매매 용이성 | "유동성 점수가 높아 매매가 원활합니다" |

### 수익률 관련 지표

| 영문 용어 | 한글 용어 | 설명 | 사용 예시 |
|----------|----------|------|----------|
| **CAGR** | **연평균성장률** | 복리 기준 연간 성장률 | "연평균성장률 15% 달성" |
| **Win Rate** | **승률** | 수익 거래 비율 | "승률 65%로 안정적입니다" |
| **Profit Factor** | **손익비율** | 총이익/총손실 비율 | "손익비율 2.5로 수익성이 좋습니다" |
| **Risk-Reward Ratio** | **위험보상비율** | 위험 대비 기대 수익 | "위험보상비율 1:3 설정" |
| **P&L (Profit/Loss)** | **손익** | 실현/미실현 손익 | "현재 손익 +500만원" |

## 📝 사용 가이드

### 코드에서의 사용 예시

```python
# 백엔드 (Python)
class 포트폴리오위험지표:
    위험가치_95: float  # VaR 대신
    위험조정수익률: float  # Sharpe Ratio 대신
    최대낙폭: float  # Max Drawdown 대신

# 프론트엔드 (TypeScript)
interface 위험지표 {
    포트폴리오위험도: number;
    위험조정수익률: number;
    최대낙폭: number;
}
```

### UI 표시 예시

```typescript
// 대시보드 표시
<div>포트폴리오 위험도: {위험도}%</div>
<div>위험조정수익률: {수익률}</div>
<div>최대낙폭: {낙폭}%</div>
```
